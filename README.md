# MyPortfolio - Static Website

A sleek, early 2000's inspired static website with a modern twist! This portfolio site features an About Me homepage and two project showcase sections: PsyAi and HealthPal.

## 🎨 Design Philosophy

This website blends the nostalgic aesthetic of early 2000s web design (think glossy buttons, gradients, and rounded corners) with modern web development practices for optimal performance and responsiveness.

## 📁 Project Structure

```
├── index.html      # Main HTML file with tab-based navigation
├── style.css       # All styling with Y2K-inspired aesthetics
├── script.js       # Tab switching functionality and animations
└── README.md       # This file
```

## 🚀 Features

- **Tab-Based Navigation**: Smooth transitions between About Me, PsyAi, and HealthPal sections
- **Responsive Design**: Works beautifully on desktop, tablet, and mobile devices
- **Early 2000's Aesthetic**: Gradient buttons, glossy effects, rounded corners with modern refinement
- **No Dependencies**: Pure HTML, CSS, and vanilla JavaScript - no frameworks needed

## 📦 Deployment to GitHub Pages

### Method 1: Using a New Repository

1. Create a new repository on GitHub (e.g., `myportfolio`)
2. Upload all files (`index.html`, `style.css`, `script.js`, `README.md`)
3. Go to **Settings** → **Pages**
4. Select the branch you want to deploy from (usually `main` or `master`)
5. Click **Save**
6. Your site will be live at: `https://[your-username].github.io/myportfolio/`

### Method 2: Using GitHub User/Organization Site

1. Create a repository named `your-username.github.io` (replace with your actual username)
2. Upload all files to this repository
3. Go to **Settings** → **Pages**
4. Select the branch and save
5. Your site will be live at: `https://[your-username].github.io/`

### Quick Deployment Commands

```bash
# Clone your repository
git clone https://github.com/YOUR_USERNAME/myportfolio.git
cd myportfolio

# Add all files
git add .

# Commit changes
git commit -m "Initial portfolio website deployment"

# Push to GitHub
git push origin main
```

## 🎨 Customization

### Changing Colors

Edit the gradient colors in `style.css`:

```css
/* Main background gradient */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* Header button gradient */
background: linear-gradient(180deg, #667eea 0%, #764ba2 100%);
```

### Updating Content

Edit the content directly in `index.html`:
- **About Me section**: Lines ~35-65
- **PsyAi section**: Lines ~67-98
- **HealthPal section**: Lines ~100-131
